# Mingzi Ye Portfolio

Fresh Kinetic Cutboard portfolio site for Mingzi Ye, a Boston-based video producer, shooter, and editor.

## Local Development

```powershell
npm install
npm run dev
```

## Production Build

```powershell
npm run build
```

## Deployment

This project includes a GitHub Pages workflow at `.github/workflows/deploy.yml`.

After pushing to a GitHub repository:

1. Open the repository on GitHub.
2. Go to `Settings > Pages`.
3. Set source to `GitHub Actions`.
4. Push to `main` to trigger deployment.

The live URL will be available from the workflow summary after deployment.
